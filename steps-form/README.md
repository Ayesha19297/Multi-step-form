Title: Multi-Step Form with React

Tech stack: React.js

This is a responsive multi-step form with three steps, ensuring data validation, error handling, and navigation controls. The form persist data to local storage and enforce sequential completion of steps.

Step 1: Personal Information (Name, Email, Phone)
Step 2: Address Information (Address Line 1, Address Line 2, City, State, Zip Code)
Step 3: Confirmation (Review all entered data)

Check the live version : https://ayesha19297-multi-step-form.netlify.app/
